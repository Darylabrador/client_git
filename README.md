# Desktop App using ElectronJs & Nodejs : Git client

## Technologies

- Electron (nodejs)

## Modules installation

> npm install

## Start the app

1. In dev mode  : npm run elecdev
2. In prod mode : npm start

## Documentations

To generate the documentation, you can use this following command : npm run doc
